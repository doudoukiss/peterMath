# peterMath：数字生命的数学实验室

## 创意来源

我想研究一个问题：简单的数学规则能不能产生像生命一样复杂的形状？

## 数学规则

作品包含几类规则系统：

1. Game of Life：每个格子根据周围 8 个邻居决定出生、存活或死亡。
2. Lenia-like field：把格子生命变成连续数值场，用邻域卷积和增长函数生成更柔软的生命形态。
3. Reaction-Diffusion：两种虚拟化学物质扩散和反应，形成斑点、波纹和迷宫结构。

## 工程实现

程序使用 Rust 开发，目标是生成 Windows 可运行程序。界面中可以调整参数、选择种子、观察指标，并导出实验结果。

## 创新点

- 把数学规则变成可操作的软件实验室。
- 同时提供 Raw Math View 和 Artistic View，说明美术效果来自数学数据。
- 通过 mass、entropy、symmetry、stability、vitality 等指标比较不同种子的表现。

## 实验结论示例

当我只改变一个参数时，图案会从稳定团块变成扩散纹理，说明复杂形态可能来自简单规则和参数之间的关系。
