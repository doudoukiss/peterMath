# peterMath：数字生命的数学实验室

## 创意来源

我想研究一个问题：简单的数学规则能不能产生像生命一样复杂、优美、可解释的形状？

## 数学规则

作品包含三类规则系统：

1. Lenia-like field：把生命格子扩展成连续数值场，用邻域卷积、增长函数和阻尼生成柔软的生命形态。
2. Reaction-Diffusion：两种虚拟化学物质扩散和反应，形成斑点、波纹和迷宫结构。
3. Game of Life：每个格子根据周围 8 个邻居决定出生、存活或死亡，并支持 RLE 图案导入导出。

## 工程实现

程序使用 Rust 开发，目标是生成 Windows 可运行程序。主要作品界面是 GPU Lenia；如果 GPU 不可用，程序会保留 CPU 参考路径。界面中可以调整参数、选择种子、绘制/擦除/盖章、观察指标，并导出实验结果。

## 解释层

作品不只展示动画，还展示数学原因：

- Raw Math View 显示原始数值场。
- Artistic View 用同一个数值场生成更有美感的视觉表达。
- Field Inspector 显示某一点的数值、变化量、梯度、卷积结果、增长响应和预测下一步。
- Kernel Lens 显示邻域核半径和核曲线。
- 指标曲线显示 mass、entropy、stability、vitality 的变化。
- Active Region 显示活跃区域、中心点和漂移方向。
- Game of Life 会识别 block、beehive、loaf、blinker、toad、beacon 和 glider，并估计振荡周期和 glider 漂移。
- Rule Variant Explorer 可以用同一个 Lenia 初始状态比较一个参数改变后的指标差异。

## 分享与证据

程序可以导出截图、参数 JSON、share-state JSON 和 evidence pack。这样评委可以看到同一个种子、同一组参数和同一帧画面之间的对应关系。

## 创新点

- 把数学规则变成可操作的软件实验室。
- 用美感吸引观众，再用检查器和指标解释美感从哪里来。
- 用已知图案检测、活跃区域和参数对照实验帮助评委看见“涌现”不是随机动画。
- 同时提供原生程序和离线网页备用演示，降低评审现场运行风险。

## 实验结论示例

当我只改变 Lenia 的 growth center 或 kernel radius 时，图案会从稳定团块变成漂移、开花或衰减状态。说明复杂形态可能来自简单规则和参数之间的关系。
