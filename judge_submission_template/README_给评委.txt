peterMath：数字生命的数学实验室

打开方式：
1. 双击 peterMath.exe。
2. 如果 Windows 出现安全提示，请选择“更多信息”→“仍要运行”。这是未签名学生软件常见提示。
3. 打开后建议先点击 Judge Mode。

备用打开方式：
如果 peterMath.exe 在评审电脑上无法启动，请打开 web_html/index.html。网页版是离线备用演示窗口，可以展示反应扩散数学规则、参数、种子、Raw Math View / Artistic View 和分享链接；正式作品仍以 peterMath.exe 为主。

建议评审路径：
1. 默认进入 Lenia-like field，观察 GPU Lenia living field。
2. 切换 Raw Math View，观察原始数学场。
3. 切回 Artistic View，比较同一数据的视觉表达。
4. 打开 Judge Mode，查看右侧 Field Inspector、Kernel Lens、指标曲线和 phase label。
5. 只改变一个参数，例如 growth center 或 kernel radius，观察图案和指标变化。
6. 点击 Export share state 或 Evidence pack，查看可复现实验 JSON、截图和 SUMMARY.md。
7. 切换 Reaction-Diffusion，调整 feed/kill。
8. 切换 Game of Life，可尝试 RLE 导入/导出。

作品说明：
peterMath 不是预制动画。每一帧都由数学规则实时计算生成，包括连续 Lenia 场、反应扩散和经典 Game of Life。作品重点是展示数学规则、参数、种子和指标如何共同生成具有生命感的形态。
