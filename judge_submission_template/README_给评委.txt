peterMath：数字生命的数学实验室

打开方式：
1. 双击 peterMath.exe。
2. 如果 Windows 出现安全提示，请选择“更多信息”→“仍要运行”。这是未签名学生软件常见提示。
3. 打开后建议先点击 Judge Mode。

备用打开方式：
如果 peterMath.exe 在评审电脑上无法启动，请打开 web_html/index.html。网页版只是备用演示窗口，用于说明反应扩散数学规则如何生成图案；正式作品仍以 peterMath.exe 为主。

建议评审路径：
1. 选择 Lenia-like field。
2. 切换 Raw Math View，观察原始数学场。
3. 点击 Run，运行约 100 步。
4. 调整 growth center 或 kernel radius，只改一个参数。
5. 观察图案和右侧指标变化。
6. 切换 Reaction-Diffusion，调整 feed/kill。
7. 点击 Export snapshot + parameters，查看导出的图像和参数文件。

作品说明：
peterMath 不是预制动画。每一帧都由数学规则实时计算生成，包括连续细胞自动机、反应扩散和经典 Game of Life 等系统。作品重点是比较不同数学规则和参数如何产生类似生命的形态。
