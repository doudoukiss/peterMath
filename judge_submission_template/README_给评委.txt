peterMath：Lenia 连续生命场计算艺术

打开方式：
1. 双击 peterMath.exe。
2. 如果 Windows 出现安全提示，请选择“更多信息”→“仍要运行”。这是未签名学生软件常见提示。
3. 打开后会自动进入“评审演示模式”，建议先完整观看约 3 分钟。

备用打开方式：
如果 peterMath.exe 在评审电脑上无法启动，请打开 web_html/index.html。网页版是离线 Lenia 备用展示窗口，可以展示连续场规则、参数、种子、数学原始图 / 艺术表达图 和分享链接；正式作品仍以 peterMath.exe 为主。

建议评审路径：
1. 先看自动演示：Lenia 是连续生命场。
2. 观察轨道场、双生命体、卷积核环、密集开花和珊瑚衰退。
3. 切换“数学原始图 / 艺术表达图”，比较同一数据的两种表达。
4. 查看右侧检查器：u、K*u、增长响应、梯度和估计下一帧。
5. 打开“主要情况”，一键载入代表性案例，不需要手动找参数。
6. 使用“规则变量对照”，只改变一个 Lenia 参数并比较指标差异。
7. 用画笔、擦除、盖章或参数滑条操作画布，确认它是实时模拟。
8. 点击“导出可复现状态”或“证据包”，查看 JSON、截图和 SUMMARY.md。

作品说明：
peterMath 不是预制动画。每一帧都由 Lenia 连续场规则实时计算生成。作品重点是展示一个简单公式如何通过卷积核、增长函数、阻尼、种子和交互，产生具有生命感的形态，并且可以被检查、比较和导出证据。
