peterMath：Lenia 数学生命教学游戏

打开方式：
1. 优先双击 START_WINDOWS.bat 或 双击运行-评委版.bat。
2. 也可以直接双击 peterMath.exe。
3. 如果 Windows 出现安全提示，请选择“更多信息”→“仍要运行”。这是未签名学生软件常见提示。
4. 打开后会进入“任务模式”。请先按左侧任务玩 3 分钟，再查看专家证据。

备用打开方式：
如果 peterMath.exe 在评审电脑上无法启动，请双击 打开备用网页.bat，或直接打开 web_html/index.html。网页版是离线 Lenia 教学游戏备用窗口，可以展示任务、连续场规则、参数、种子、数学原始图 / 艺术表达图 和分享链接；正式作品仍以 peterMath.exe 为主。

建议评审路径：
1. 看见 `1 选工具 / 2 选任务 / 3 点生命场` 后，从“唤醒生命场”开始。
2. 让连续场运行并观察目标反馈。
3. 进入“塑造生命”，用画笔或盖章改变场。
4. 进入“半径挑战”，只改变一个规则参数并观察指标变化。
5. 进入“证明同一数据”，切换“数学原始图 / 艺术表达图”，再用检查器查看同一点。
6. 进入“生成证据报告”，点击“导出可复现状态”或“证据包”。
7. 如需学生讲解，再打开“自动讲解”观看完整 Lenia 叙事。

作品说明：
peterMath 不是预制动画，也不是只给专业人士看的技术面板。每一帧都由 Lenia 连续场规则实时计算生成。作品重点是把一个简单公式做成可玩的教学任务，让评委先操作，再通过数学卡片、指标和导出证据理解卷积核、增长函数、阻尼、种子和交互如何产生生命感形态。
