peterMath：数字生命的数学实验室

打开方式：
1. 双击 peterMath.exe。
2. 如果 Windows 出现安全提示，请选择“更多信息”→“仍要运行”。这是未签名学生软件常见提示。
3. 打开后会先看到“三系统总览”，建议先观察三块画面，再选择“进入实验”。

备用打开方式：
如果 peterMath.exe 在评审电脑上无法启动，请打开 web_html/index.html。网页版是离线备用演示窗口，可以展示数学规则、参数、种子、数学原始图 / 艺术表达图 和分享链接；正式作品仍以 peterMath.exe 为主。

建议评审路径：
1. 先看三系统总览：生命游戏、反应扩散、连续生命场 Lenia。
2. 对比三块画面的公式、阶段、活跃度和最近变化量。
3. 点击任一“进入实验”，右侧使用中文预设优先控制。
4. 切换“数学原始图 / 艺术表达图”，比较同一数据的两种表达。
5. 打开“评审讲解模式”，查看检查器、卷积核、指标曲线、活跃区域和阶段结论。
6. 在 Lenia 中使用“规则变量对照”，只改变一个参数并比较指标差异。
7. 在生命游戏中观察静态结构、振荡周期和滑翔机追踪。
8. 点击“导出可复现状态”或“证据包”，查看 JSON、截图和 SUMMARY.md。

作品说明：
peterMath 不是预制动画。每一帧都由数学规则实时计算生成，包括连续 Lenia 场、反应扩散和经典 Game of Life。作品重点是展示数学规则、参数、种子和指标如何共同生成具有生命感的形态。
