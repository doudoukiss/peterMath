# peterMath Web Teaching Fallback

This folder contains a standalone Lenia teaching-game fallback for judges to open if the Windows executable cannot run on a specific computer.

Open:

```text
index.html
```

No server, Node, npm, Python, Rust, internet connection, or browser extension is required.

## Purpose

The native `peterMath.exe` remains the primary submission. This web fallback is a backup playable window that preserves the same judging story:

- judges start with missions instead of a blank technical panel;
- a continuous Lenia field is generated by a local mathematical rule;
- parameters change the pattern;
- deterministic seeds make experiments repeatable;
- Raw Math View, Artistic View, and Life Highlight View show the same underlying field differently;
- metrics summarize the evolving field;
- export/share actions provide evidence after play.

## Sharing

The fallback supports URL hash sharing. A link can preserve preset, seed, Lenia parameters, view style, and speed:

```text
index.html#preset=orbital&seed=1001&radius=9&center=0.31&width=0.052&damping=0.003&style=artistic&steps=1
```

Use `Copy share link` inside the page to generate the current link. Use `Export snapshot` to download the visible canvas as a PNG.

The browser version is intentionally smaller than the native GPU app. It exists as an emergency/public Lenia teaching game, not as a replacement for the native app.
