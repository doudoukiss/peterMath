peterMath
Lenia 备用展示

这是离线网页 fallback。正式作品是原生 GPU Lenia 程序；这里用于证明同一类连续场规则可以在浏览器里实时运行。

u_next = clamp(u + dt*(G(K*u) - damping*u), 0, 1)

控制

peterMath 连续生命场 Lenia
点击或拖动画布可注入生命量。参数变化会立即重置当前规则并实时重算。

看什么

观察同一公式如何从不同初始条件生成漂移、耦合、尺度变化、湍动和衰退。

变量含义

u 是当前生命量；K*u 是邻域卷积；G 是增长函数；damping 是阻尼。
