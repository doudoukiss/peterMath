peterMath current context
=========================

The project has converged to one competition identity: Lenia as a living mathematical field.

Goal
----
Create an elegant computational artwork that persuades judges through beauty and mathematical clarity. The artwork should show that local field rules can generate coherent form, motion, instability, decay, and evidence that can be inspected and reproduced.

Core idea
---------
Each point in the Lenia field stores a scalar value `u`. On each step, the app computes a local weighted neighborhood `K*u`, maps it through a growth response `G`, subtracts damping, and clamps the result:

```text
u_next = clamp(u + dt*(G(K*u) - damping*u), 0, 1)
```

The artistic view must be derived from the same scalar field: value, gradient, contour bands, and frame delta. There should be no decorative particles and no unrelated visual systems.

Native direction
----------------
- Native Rust app is primary.
- GPU Lenia is the main viewing path.
- CPU Lenia remains fallback/reference for metrics, inspector, previews, and exports.
- Chinese UI is primary.
- The app opens into a guided Lenia show mode, then allows manual experimentation.

Implemented pillars
-------------------
- Lenia presets: orbital field, twin organisms, coral drift/fading, kernel ring, sparse soup, dense bloom.
- Brush and stamp editing.
- Undo/redo.
- Raw and Artistic views from the same field.
- Inspector for value, previous value, delta, gradient, convolution, growth, and estimated next value.
- Metrics: mass, entropy, symmetry, stability, vitality, active region, phase.
- Rule variant comparison for radius, growth center, growth width, and damping.
- PNG/JSON snapshot export, share-state JSON, and evidence packs.
- Preview generation and an offline `web_html/` fallback.

Show mode direction
-------------------
The judge should not need to tune parameters first. The app should guide them through a concise Lenia story:

1. Lenia is a continuous living field.
2. Local convolution creates drift.
3. Two structures influence each other through the same field.
4. Kernel radius controls visual scale.
5. Dense fields show growth, saturation, and turbulence.
6. Damping and boundary conditions create fading.
7. Exported evidence records the same visible state.

Repository rule
---------------
Keep the repository focused. Do not reintroduce secondary simulation systems unless the competition strategy changes and the judging story remains clearer than the current Lenia-only direction.
